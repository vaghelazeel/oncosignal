# OncoSignal

Machine learning for pharmacovigilance signal detection in oncology using FDA FAERS data.

## Status

Phase 0 — Project planning complete. Phase A — data engineering in progress.

## Project goals

This project applies machine learning and natural language processing to FDA Adverse Event Reporting System (FAERS) data to detect safety signals for anticancer drugs. It combines traditional pharmacovigilance disproportionality methods (PRR, ROR, IC) with supervised and unsupervised machine learning.

## Tech stack

- Python 3.12
- pandas, numpy, scipy
- scikit-learn, xgboost, shap
- spaCy, BERTopic, sentence-transformers
- matplotlib, seaborn, plotly
- Jupyter
